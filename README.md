# eCommerce Progressive Web Application

MongoDB back-end database<br>
Node (Express + Mongoose) API layer<br>
React front-end client

# Installation

`git clone https://github.com/MihaiGitHub/eCommerce.git`

# Running the API layer

`cd api`<br>
`npm install`<br>
`npm start`

# Running the client

`cd client`<br>
`npm install`<br>
`npm start`
